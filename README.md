# ACID KHUTIR — Neon Psychedelic Cyber‑Folk Defense Game

ACID KHUTIR — це 2D Neon Psychedelic Cyber‑Folk гра на Phaser 3, де ви захищаєте українське село Ланчин від вірусу Mausoleum 2.1 — інформаційної аномалії, що перетворює радянських бюрократів на кислотних зомбі.

## 🎮 Жанр
- Castle Defense
- Roguelike Progression
- Bullet‑Hell Elements
- Acid Neon Art Style

## 🧠 Основна ідея
Ви граєте за Сергія — кібер‑тракториста з енергоканоном, який захищає Хату від нескінченних хвиль зомбі‑клерків, архіваріусів та інспекторів. Кожні 5 хвиль ви обираєте перки, створюючи унікальні білди.

## ⚡ Особливості
- Neon Psychedelic Cyber‑Folk візуальний стиль
- 3 рівні Хати (Traditional → Reinforced → Cyber‑Sich)
- 20+ хвиль ворогів
- 3 перки з синергіями (Golden Coupon, Radioactive Beet, Iron Seal)
- Бос: Comrade Vakhtersha
- Динамічна музика (acid folk)
- Глітч‑ефекти, hit stop, camera shake

## 🚀 Встановлення та запуск
```bash
npm install

npm run build    # продакшн-збірка
npm run preview  # попередній перегляд збірки
```
