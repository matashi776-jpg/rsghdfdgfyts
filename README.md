# 🔥 ACID KHUTIR — Оборона Ланчина

> **Neon Psychedelic Cyber-Folk 2D Tower Defense**  
> Phaser 3 + Vite | WebGL | GitHub Pages

Захищай Ланчин від кислотних зомбі-бюрократів, заражених вірусом Mausoleum 2.1!

---

## 🚀 Швидкий старт

```bash
npm install
npm run dev       # http://localhost:8080
npm run build     # production build → dist/
```
