# AI_PROMPTS.md — Neon Psychedelic Cyber-Folk Art Generation Prompts

> **Rule:** Always use these exact prompts when generating assets with Gemini / Midjourney / DALL·E.  
> Do **not** change the style, palette, or line-thickness between sessions.

> See also: `ASSET_MANIFEST.json` for the authoritative list of every required file.

## 📐 TECHNICAL REQUIREMENTS (all assets)

| Property          | Standard                                                     |
|-------------------|--------------------------------------------------------------|
| Format            | PNG (32-bit with alpha, except backgrounds which are RGB)    |
| Background        | **Transparent** (sprites/icons/FX) · **Opaque** (locations, cutscenes, comics) |
| Size — characters | 2048 × 2048 px                                               |
| Size — items/perks| 256 × 256 px (items) · 128 × 128 px (perk icons)            |
| Size — FX (large) | 512 × 512 px                                                 |
| Size — FX (small) | 128 × 64 px (bullet) · 256 × 256 px (hit/explosion)         |
| Size — UI         | Varies per element (see per-asset spec above)                |
| Size — locations  | 1280 × 720 px                                                |
| Size — symbols    | 128 × 128 px                                                 |
| Size — cutscenes  | 1280 × 720 px                                                |
| Size — comics     | 800 × 600 px                                                 |
| Outline           | Pure black (#000000), **4–6 px**                             |
| Glow              | **Baked into PNG** — never via shader or layer effect        |
| Style             | **Cel-shading only** — no realism, no soft blur              |
| Animations        | **6–12 frames** per action (see per-asset spec)              |
| Palette           | Electric Blue `#00BFFF` · Neon Pink `#FF00AA` · Toxic Green `#39FF14` · Ultra-Violet `#7F00FF` |
