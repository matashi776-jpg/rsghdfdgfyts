# ART BIBLE — ACID KHUTIR

## Стиль
**Neon Psychedelic Cyber-Folk**

Поєднання:
- Радянський конструктивізм
- Українська народна вишивка та писанки
- Acid/rave естетика 90-х
- Cyberpunk neon glow
- Pixel-adjacent cel-shading

---

## Правила стилю

1. ЗАВЖДИ використовуй товсті чорні обводки (min 4px)
2. ТІЛЬКИ cel-shading — жодного реалізму, жодного blur
3. Кожен персонаж повинен мати Ukrainian folk pattern деталь
4. Neon glow = обов'язковий для кожного спрайта
5. Прозорий фон (alpha channel) для всіх спрайтів
