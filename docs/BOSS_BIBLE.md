# BOSS BIBLE — ACID KHUTIR

## Вахтерша — Mausoleum Protocol 2.1

### Концепція
Бетонний моноліт + багаторука радянська вахтерша + глитч-монітор замість голови.
Уособлює всю систему радянської бюрократії.

### Фаза 1 (100% — 50% HP)
- **Speed:** 28 px/s
- **Атака 1:** Удар рукою-штампом (зона 120px, стан: оглушення 1.5s)
- **Атака 2:** Кидає "довідку" (проджектайл 200px/s)
- **Інтервал атак:** 2.5s
- **Тінт:** Default

### Фаза 2 (< 50% HP)
- **Speed:** 48 px/s (+71%)
- **Атака 1:** Розгін (спринт до гравця)
- **Атака 2:** АоЕ штамп (зона 200px)
- **Атака 3:** Запускає 3 клерки одночасно
- **Тінт:** 0xff0055 (червоний глитч)

### Нагорода за перемогу
- 150 золота
- Розблоковує кінцівку гри
- Meta-currency: +10

- Висота: ~200px (140px в грі)
