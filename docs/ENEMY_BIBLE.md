# ENEMY BIBLE — ACID KHUTIR

## Зомбі-клерк (enemy_zombie_clerk)
- **Роль:** Базовий ворог, з'являється зграями
- **HP формула:** Base × 1.0
- **Speed формула:** Base × 1.0
- **Силует:** Тонкий, нахилений, папери в руках
- **Кольорова схема:** Neon Pink слиз, блискучі очі
- **Поведінка:** Прямолінійний рух до хати, іноді вигукує бюрократичні фрази
- **Смерть:** Розсипається папером + кислотний сплеск

## Архіваріус (enemy_archivarius)
- **Роль:** Важкий ворог-танк
- **HP формула:** Base × 2.2
- **Speed формула:** Base × 0.55
- **Силует:** Квадратний, стопки папок
- **Кольорова схема:** Toxic Green, LED очі
- **Поведінка:** Повільний рух, блокує кулі папками (30% шанс)
- **Смерть:** Розсипається стопками папок + зелений дим

## Інспектор (enemy_inspector)
- **Роль:** Швидкий скірмішер
- **HP формула:** Base × 0.65
- **Speed формула:** Base × 1.8
- **Силует:** Середній, з величезною печаткою
- **Кольорова схема:** Neon Pink, документарна броня
- **Поведінка:** Зигзаг-рух, може оглушити гравця печаткою
- **Смерть:** Розриває документи + рожевий спалах
