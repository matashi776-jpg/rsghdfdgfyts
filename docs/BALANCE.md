# BALANCE.md — ACID KHUTIR

## Формули

### Ворог HP
```
HP = 60 + Wave × 15 + log(Wave + 1) × 10
```

| Хвиля | HP  |
|-------|-----|
| 1     | 86  |
| 3     | 121 |
| 5     | 155 |
| 7     | 187 |
| 10    | 230 |

### Швидкість ворога
```
Speed = 40 + Wave × 2
```

| Хвиля | Speed |
|-------|-------|
| 1     | 42    |
| 5     | 50    |
| 10    | 60    |

### Золото
```
Gold = 5 × (1 + Wave × 0.05)
```

| Хвиля | Gold/kill |
|-------|-----------|
| 1     | 5.25      |
| 5     | 6.25      |
| 10    | 7.50      |
