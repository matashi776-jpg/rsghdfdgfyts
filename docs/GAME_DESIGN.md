# GAME DESIGN DOCUMENT — ACID KHUTIR

## Жанр
- 2D Defense
- Roguelike progression
- Bullet-hell elements

## Ціль
Захистити хату від хвиль ворогів вірусу Mausoleum 2.1.

---

## Механіки

| Механіка         | Опис                                             |
|------------------|--------------------------------------------------|
| Стрільба         | WASD рух + автоціль / cursor aim                 |
| Перки            | 3 варіанти між хвилями                           |
| Хвилі            | 10 хвиль, 10-та — бос                           |
| Бос              | Вахтерша з 2 фазами                             |
| Мета-прогресія   | Золото між ранами для покращень                  |
| FX               | Глитч, hit-stop, camera shake, neon particles   |

---

## Формули балансу

### HP ворога
```
HP = 60 + Wave × 15 + log(Wave + 1) × 10
```

### Швидкість ворога
```
Speed = 40 + Wave × 2
```

### Нагорода золотом
```
Gold = 5 × (1 + Wave × 0.05)
```

### Кількість ворогів на хвилю
```
Count = 3 + floor(Wave × 1.5)
```
