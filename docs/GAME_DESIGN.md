# GAME DESIGN DOCUMENT — ACID KHUTIR

## 1. Core Fantasy
Ви — Сергій, кибер-тракторист з Прикарпаття, який захищає свій кибер-хутір від кислотних зомбі-бюрократів, заражених вірусом Mausoleum 2.1.

## 2. Genre
- 2D Defense
- Roguelike Progression
- Bullet-Hell Elements
- Neon Psychedelic Cyber-Folk Art Direction

## 3. Player Abilities
- Move (WASD)
- Shoot (LMB)
- Special (після хвилі 5)
- Perks (кожні 5 хвиль)

## 4. Level Structure
Level 1 складається з:
- 3 хвиль ворогів
- 1 міні-боса
- 1 NPC (Бабця-знахарка)
- 1 хати (Khata Level 1)
- 1 арени

## 5. Wave Breakdown
Wave 1:
- 6 Zombie Clerks

Wave 2:
- 8 Zombie Clerks
- 2 Archivarius

Wave 3:
- 10 Zombie Clerks
- 3 Archivarius
- 1 Inspector

Mini-Boss:
- Mini-Vakhtersha

## 6. Victory Condition
Перемога після вбивства міні-боса.

## 7. Failure Condition
Хата зруйнована або Сергій помер.

## 8. Rewards
- Золото
- Перки
- Мета-прогресія

## 9. UI Elements
- HP Bar (rushnyk pattern)
- Wave Counter (pysanka frame)
- Ammo Indicator (neon glyphs)
- Start Button ("СТАРТ")
